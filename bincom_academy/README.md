# Bincom Academy Cybersecurity Labs

This repository documents hands-on cybersecurity labs aligned with the
BINCOM Academy Cybersecurity Curriculum (Version 1.1). Documentation and evidence for completing the Bincom Academy Cybersecurity Curriculum.

**Curriculum Overview:** 8-day hands-on program covering network security, vulnerability assessment, web security, incident response, cloud IAM, CTF, and testing.

**Important Warnings:**
- All labs run in isolated VMs (VirtualBox/VMware host-only adapter).
- No real-world attacks or unauthorized scanning.
- AWS: Use free tier; delete resources after labs to avoid costs.
- Ethical use only; document assumptions and limitations.

**Repo Contents:**
- Detailed setup guides and commands per day.
- Report templates with placeholders for my findings/screenshots.
- Screenshots in `/screenshots/`

**Setup Prerequisites:**
- VirtualBox or VMware Workstation/Player.
- Host machine: 16GB+ RAM, 100GB+ free disk.
- Kali Linux ISO (official: https://www.kali.org/get-kali/).

## Objective
The objective is to demonstrate:
- Practical security engineering skills
- Attack detection and exploitation
- Defensive controls and remediation
- Professional security reporting

All activities were conducted in controlled lab environments for educational purposes only.
