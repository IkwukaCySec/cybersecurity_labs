# Phishing Awareness Quiz

1. This email is from "it@tbfc.com" but Return-Path is gmail.com. Safe?  
   → **No — Spoofed sender**

2. Link says microsoft.com but has Cyrillic о. Safe?  
   → **No — Homoglyph attack**

3. "Urgent password update required by Dec 24". Likely real?  
   → **No — Creates urgency**

4. Best action when suspicious?  
   → **Hover link, check URL, report to SOC**

**Score 4/4? You're phishing-resistant!**
